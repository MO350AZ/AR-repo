# AR-repo (Filtered Mirror)

This repository is a **filtered mirror** of the official Keiyoushi extensions repository.

## Source
Original repository:
https://github.com/keiyoushi/extensions

Original index:
https://raw.githubusercontent.com/keiyoushi/extensions/repo/index.min.json

## What this repo does
- Automatically fetches the original index from Keiyoushi
- Removes NSFW entries only
- Does NOT modify source names, packages, or APKs
- Does NOT re-host APK files
- Preserves full credit and ownership to the original authors

## Disclaimer
All extensions, APKs, and source code belong to their respective owners.
This repository only provides a filtered index for convenience.
