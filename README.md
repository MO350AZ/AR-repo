# AR-repo (Filtered Mirror)

This repository is a filtered mirror of the official Keiyoushi extensions index.

## Source
- Official repo: https://github.com/keiyoushi/extensions
- Official index: https://raw.githubusercontent.com/keiyoushi/extensions/repo/index.min.json

## What this repo does
- Fetches the official index automatically
- Removes NSFW entries only
- Does NOT re-host APK files
- Preserves original names, packages, and ownership

## Disclaimer
All extensions belong to their respective owners.
This repository only provides a filtered index for convenience.
